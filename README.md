WASp
====

## How to run this

1. Install it

   ```bash
   pip install git+https://github.com/LoicGrobol/wasp.git
   ```

2. Run it on your conll file

   ```bash
   wasp myfile.conll
   ```

3. See the help for more details

   ```bash
   wasp --help
   ```
